# CoffeeScript guidelines

## Text editor settings
- Tab size: 2 spaces
- Line size: < 120 signs
- Empty last line
- Remove trailing spaces

## Use
- Simple quotes
- Namespaced global variables
- Browserify
- Lint files
- Source map
- CoffeeDoc comments

## Don't
- Implicit return
- Nested functions without ~parentheses~
- Global variables
